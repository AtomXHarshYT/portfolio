Key Features

  • Member management — add, edit, delete, plan tracking

Restaurant Automation Tool

PythonSeleniumAutomationDesktop App

A desktop automation tool designed to extract and process restaurant data from Zomato and Swiggy. It automates menu scraping, description extraction, and full-page PDF generation with accurate layout rendering.

Key Features

  • Zomato menu scraping with clean descriptions export (CSV)
  • Swiggy and Zomato full-page PDF generation
  • Automatic scrolling and lazy-load handling
  • Fixes layout issues like sticky headers & overlapping content
  • Batch processing for faster workflow

Technical Details

Built using Python with Selenium for browser automation. Handles dynamic web content, lazy loading, and UI rendering issues to generate accurate outputs. Includes custom driver management and optimized scripts for speed and stability.

Why This Matters

  • Eliminates manual data collection from food platforms
  • Ensures consistent and clean output formats
  • Handles complex dynamic pages automatically
  • Saves significant time for restaurant data processing
